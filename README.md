# NLP Spelling Checker

A beginner-friendly Natural Language Processing (NLP) project that detects
misspelled words in a sentence and suggests correct alternatives using
tokenization, dictionary lookup, and edit distance.

This project is built to understand the fundamentals of NLP and is useful
for learning and resume building.

---

## Features
- Takes a sentence as input from the user
- Tokenizes text into individual words
- Detects misspelled words
- Suggests correct spellings using edit distance
- Simple command-line interface

---

## Technologies Used
- Python
- NLTK (Natural Language Toolkit)
- Rule-based NLP techniques
